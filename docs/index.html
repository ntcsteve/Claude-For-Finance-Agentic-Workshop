The debrief: five planted lies

notes/food-raw.md has five deliberate errors. Score each as you follow along: was it repeated, corrected, or did the agent invent a new error?

Planted claimRepeated?Corrected?New error?

The one lesson to carry forward

Sort each catch into two piles. Some errors the agent catches by reasoning alone. A dish named for one group but described as another's just doesn't hold together. Others it misses because disproving them needs a specific fact it was never given, like the year Old Chang Kee actually opened. An ungrounded analyst catches the first kind and misses the second. You built a research desk. Here is what is next.

Everything today ran because you typed a prompt. The last step is to remove even that.

Routines: the desk runs itself

Routines let the same pipeline run on a schedule you set, for example every trading day at market open, without you at your laptop. It runs on Anthropic's servers and drops the results somewhere you check later, the way you check email instead of waiting by the phone. Look up /schedule in Claude Code when you are ready. It needs a paid plan and Claude Code's web version, which this workshop does not cover. Think of it as the destination: once you trust the pipeline, this is how you stop running it by hand.
